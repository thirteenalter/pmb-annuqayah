# PMB Universitas Annuqayah - Sistem Informasi Penerimaan Mahasiswa Baru

Sistem Informasi **Penerimaan Mahasiswa Baru (PMB)** Universitas Annuqayah adalah platform berbasis web yang dirancang untuk mendigitalisasi dan mempermudah proses pendaftaran calon mahasiswa. Dibangun menggunakan **Laravel 11** dan **Laravel Breeze**, sistem ini menawarkan antarmuka modern, responsif, dan keamanan yang cukup.
